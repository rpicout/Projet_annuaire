LE PROJET

    Le besoin :

  • Créer un outil pour rechercher un établissement scolaire selon le critère suivant : le code postal.
  • Permettre de visualiser sous forme d'un tableau la liste des établissements scolaires.
  • Permettre d'afficher la fiche d'un établissement.
  • Permettre de modifier la fiche d'un établissement.

    Les données :

  • Les données ont été récupérées via l'Open Data de l'Eduction Nationale.
